# AP+Web 配置功能实现总结

## 功能概述

本次更新为 WeWeather 项目添加了基于 AP+Web 的配置功能，用户可以通过浏览器配置设备参数，无需串口连接。

## 实现的功能

### 1. ✅ RXD 引脚检测
- 在启动时检测 RXD 引脚（GPIO-3）状态
- 如果 RXD 被拉低，自动进入 Web 配置模式
- 引脚配置为上拉输入模式，确保稳定检测

### 2. ✅ RTC 唤醒控制
- 进入配置模式前自动禁用 RTC 定时器和中断
- 防止配置过程中被 RTC 唤醒导致重启
- 确保配置过程的稳定性

### 3. ✅ AP 模式启动
- 自动启动 WiFi AP 模式
- 默认 SSID: weweather
- 无密码（可根据需要添加）
- 默认 IP 地址: 192.168.4.1

### 4. ✅ 屏幕信息显示
- 在墨水屏上显示配置信息
- 显示 AP SSID 和 IP 地址
- 显示访问提示信息

### 5. ✅ Web 配置界面
- 响应式 HTML 界面
- 支持的配置项与串口配置完全一致
- 配置成功后自动重启

## 使用方法

1. 将 RXD 引脚（GPIO-3）接地
2. 启动设备
3. 连接到 WiFi: weweather
4. 访问 http://192.168.4.1
5. 填写配置信息并保存
